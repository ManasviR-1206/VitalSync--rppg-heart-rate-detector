🚀 Overview

VitalSync is an AI-powered contactless vital monitoring system that uses remote photoplethysmography (rPPG) and computer vision to estimate human vital signs through a standard camera.
It analyzes facial video in real time to measure heart rate, SpO₂, respiration, and stress levels.
<br> </br>
Features
📡 Real-time heart rate monitoring (BPM)
🩺 SpO₂ (oxygen level) estimation
🌬 Respiration rate detection
😌 Stress level analysis
🎥 Live facial tracking using camera
📊 Interactive dashboard with visualization
📄 Export reports (PDF)
<br></br>
Tech Stack
Frontend: HTML, CSS, JavaScript
Backend: Node.js, Express
Database: MongoDB
AI/Processing: rPPG (video-based signal extraction)
